# Human Authorship Logging Protocol

A workflow to capture and document human decisions in AI-assisted work, ensuring creative control is clear and verifiable.

## Key Features

- Records strategic choices at the end of each session, day, milestone, and project completion  
- Tracks constraints, information added, preference selections, and structural adjustments  
- Works across AI models and long projects, reflecting real professional workflows  
- Generates a verifiable record suitable for legal, ethical, or organizational review
